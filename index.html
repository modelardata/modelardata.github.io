What is ModelarData?

At ModelarData we apply state-of-the-art research to improve data transfer, advanced analytics, and machine learning for industrial IoT. We are currently developing ModelarDB, an enterprise-grade open-source time series management system that integrates with existing distributed data processing and storage infrastructure to provide fast analytics for vast data sets of time series from industrial IoT.

Model-based Time Series Management System

By representing time series as models, ModelarDB provides high query performance and very low storage cost for vast data sets.

End-to-End Support for Industrial IoT workloads

By running on both the edge, in the data center, and in the cloud, ModelarDB is an end-to-end solution for IoT workloads.
